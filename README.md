📚 Планировщик учебных сессий – Multi‑Language Implementation
Тестовый репозиторий с реализацией консольного планировщика учебных сессий на 7 языках программирования.
Приложение помогает студентам организовать учебный процесс: создавать, отслеживать и анализировать учебные сессии по предметам.

🎯 Что такое Планировщик учебных сессий?
Умный помощник для эффективной подготовки к занятиям. Вы записываете каждую учебную сессию: предмет, тему, дату и время начала, продолжительность, приоритет. Приложение даёт вам полную картину вашей учебной нагрузки.

Основные возможности:
Добавление сессии – указать предмет, тему, дату/время, длительность (в минутах), приоритет (низкий, средний, высокий).

Просмотр сессий – все, по предмету, за сегодня, по статусу (запланирована, выполнена, пропущена).

Отметка выполнения – сессия может быть отмечена как выполненная или пропущенная.

Редактирование – изменение любых полей.

Удаление сессии.

Статистика – общее количество сессий, выполненные, общая затраченная время по предметам, средняя продолжительность.

Сохранение / загрузка данных в формате JSON (или эквивалент) для переносимости.

Цветной вывод (где поддерживается) для наглядности.

✨ Особенности
Интерактивное меню с выбором действий.

Валидация ввода даты/времени (формат ГГГГ-ММ-ДД ЧЧ:ММ).

Автоматическое определение статуса сессии (запланирована, выполнена, пропущена) на основе текущего времени (опционально).

Фильтрация по предмету, статусу, дате.

Расширенная статистика: по предметам, по дням недели.

Данные сохраняются в файл sessions_data.json (или эквивалент).

Загрузка данных при старте.
